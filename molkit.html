Molkit¶

Molkit is module containing a set of functions to manipulate molecules.

It allows easily building molecules from smiles, for which 3D coordinates are generated automatically. When specifying a forcefield for geometry refinement, please note that Ebejer et al. 2012 suggested that the MMFF forcefield is better for small molecules with no or a single rotable bond. For larger molecules UFF appears better.

Molkit enables modifying existing molecules using reaction smarts.

There is also a function to partition a protein into capped fragments for MFCC calculations.

qmflows.molkit.from_sequence(sequence, nconfs=1, name=None, forcefield=None, rms=0.1)[source]¶

Generates plams molecule from a peptide sequence. Includes explicit hydrogens and 3D coordinates.

Parameters:
  • sequence (str) – A peptide sequence, e.g. ‘HAG’
  • nconfs (int) – Number of conformers to be generated
  • name (str) – A name for the molecule
  • forcefield (str) – Choose ‘uff’ or ‘mmff’ forcefield for geometry

optimization and ranking of comformations. The default value None results in skipping of the geometry optimization step. :parameter float rms: Root Mean Square deviation threshold for removing similar/equivalent conformations. :return: A peptide molecule with hydrogens and 3D coordinates or a list of molecules if nconfs > 1 :rtype: plams.Molecule or list of plams Molecules
